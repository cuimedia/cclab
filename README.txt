Usage:
1) Create a Shopify public app with Shopify CLI (Remix template is fine).
2) Generate a Theme App Extension, then replace its folder contents with this package.

Where to put:
Place the 'whatsapp-float-extension' folder inside your app at:
extensions/whatsapp-float/

Files:
- extension.config.yml
- snippets/whatsapp-float.liquid
- assets/whatsapp-float.js
- assets/whatsapp-float.css

Enable in Online Store: Customize > App embeds > WhatsApp Floating Button
